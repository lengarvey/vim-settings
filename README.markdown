My VIM setup
===========

To install
----------

```
cd ~
git clone https://github.com/lengarvey/vim-settings.git .vim
ln -s .vim/.vimrc
```

Finished :D

To use NERDTree
---------------
Press F2

To use snipmate
---------------
<tab> after typing the snippet

To use supertab
---------------
shift-<tab> to auto complete

Have fun!
